MySQL: тип SERIAL 
BIGINT UNSIGNED NOT NULL AUTO_INCREMENT UNIQUE
Значения типа SERIAL являются ненулевыми, автоинкрементными, уникальными, являющимися разновидностью типа BIGINT.
В таблице может быть только один такой столбец, который должен быть проиндексирован. 

PostgreSQL: тип SERIAL 
При добавлении записи в таблицу, значения полей SERIAL будут присваиваться автоматически как порядковые целые числа. На значения типа SERIAL накладывается только одно ограничение — они не могут быть NULL. Таблицы, содержащие поля с этим типом данных, нельзя удалять напрямую. Сначала нужно дать команду DROP SEQUENCE, а только после этого давать команду DROP TABLE.

SQLight: 
Тип отсутствует. Для создания автоинкрементного поля используется модификатор типа AUTOINCREMENT. Команда для создания автоинкрементного поля:  INTEGER PRIMARY KEY AUTOINCREMENT.
